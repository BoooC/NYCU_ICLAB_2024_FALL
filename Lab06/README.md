## Tips
- 先分析行列式運算公式
- 算行列式的時候可以先算好總共需要多少bit
- 盡量共用乘法加法器，大顆的乘法器也可以用來做bit數比較小的乘法運算，例如一個20-bit的乘法器可以用來算10-bit的乘法
- 可以把共用的乘法加法器寫成sub-module(像Lab04使用IP那樣)以確保有共用到
- 有時候過度共用硬體反而面積會變大 (選擇輸入的MUX太大)
- 有時候多開一些bit數的面積會反而比較小
- 有時候加上一些無效的邏輯面積反而會比較小
